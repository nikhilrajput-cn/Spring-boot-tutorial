# Student Management (student-Mg)

Small Spring Boot project for managing students. This repository contains a basic Maven-based Spring Boot application.

## Quick start

1. Build: `mvnw clean package` (Windows) or `./mvnw clean package` (UNIX)
2. Run: `mvnw spring-boot:run` or run the `StudentMgApplication` from your IDE
